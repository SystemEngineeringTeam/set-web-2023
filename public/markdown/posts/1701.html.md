# 概要
webページ発表会は与えられたテーマに沿ったwebページを個人で制作して発表し合う会
レベルが2つに分けられて、それぞれのレベルで別のテーマのものを作ってもらいました。
レベル1
- テーマ：自分の出身地紹介
- 使う技術：HTML・CSS
    - ノーコードでの制作は禁止

レベル2
- テーマ：自己紹介(ポートフォリオ)
	- 発表してくれた方はそのサイトへのリンクをシス研webサイトの自己紹介ページに貼る
- 使う技術：なんでも良い(Reactなどの使用も可)
    - ノーコードでの制作は禁止

事前に行っていたwebページ勉強会で学んだことをもとにそれぞれテーマに合わせたwebページを作成してもらいました。
HTML,CSS,JSを学びたい1,2年生を対象としました。
