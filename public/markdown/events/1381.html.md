# お知らせ欄
このフォルダの記事はサイトトップページのお知らせ欄に表示されます

# 書き方
**基本的にテンプレートを使用してください**

## #at:
投稿日
基本的に触る必要はありません

ex: `#at:2024-04-10`

## #deadline:
締切日
この日まで掲載されます

ex: `#deadline:2024-05-10`

## #category:
カテゴリーです
以下などを参考にしてください
- イベント
- お知らせ
- 重要

ex: `#category:イベント`
